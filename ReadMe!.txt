------------------------------------- Set-Up -----------------------------------------------

1. Extract the folder 'PIV_Self_Group_7' to a convenient location. This is the program's 
   main directory. 

2. Place image pairs in the folder titled 'Images'. The image pairs must be named following
   the 'B000-' format.

3. Place mask files in the folder named 'Mask'.

------------------------------------ Execution ---------------------------------------------

1. Run PIV_Main. Input parameters and select image pair range. The results are written to a 
   '.mat' file in the 'Results' folder.

2. Run PIV_Stat if more than one image pair is processed.
